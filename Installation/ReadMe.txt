Installation Operation Instructions

1. Make sure both all 3 USB ports are plugged in to the side of the machine
2. Ensure that the bluetooth is turned on
3. Open PowerMate Application (in the dock) and confirm that the device is connected
4. Open System Control App in the folder this ReadMe is located
5. The visualization should be running correctly
